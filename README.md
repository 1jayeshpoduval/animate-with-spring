# Animate with Spring
The easiest way to configure tasteful spring animations.

This is a developer tool built for designers, front-end engineers, and motion enthusiasts who want to craft beautiful spring animations without the hassle of guessing stiffness, damping, or bounce values. Whether youâ€™re using Motion (prev. Framer Motion) or SwiftUI, this tool helps you configure and copy spring animations faster.

## What you can do

### 1. Choose from a wide variety of finely tuned spring presets
Explore a curated library of spring animation presets that feel great out of the box. Choose your favourite and instantly copy the code in either Framer Motion or SwiftUI format.

### 2. Create your own spring animation
Tweak mass, stiffness, damping, bounce, and duration values to create your own spring animation. See real-time visual feedback and generate production-ready code instantly.

### 3. Learn how to animate tastefully
Deep dive into the fundamentals of spring physics. Understand what stiffness and damping actually do, and learn the principles behind tasteful UI animation with helpful examples and best practices. Includes curated reading material for further exploration.

## Why use Animate with Spring?
* Save time when prototyping or refining your product's motion.

* Copy clean, production-ready Motion and SwiftUI animation code.

* Learn what makes a great spring animation from a design and UX perspective.
